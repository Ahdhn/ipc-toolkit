template<typename DerivedEA0, typename DerivedEA1, typename DerivedEB0, typename DerivedEB1, typename DerivedGrad>
void ipc::edge_edge_distance_gradient(const Eigen::MatrixBase<DerivedEA0> &ea0, const Eigen::MatrixBase<DerivedEA1> &ea1, const Eigen::MatrixBase<DerivedEB0> &eb0, const Eigen::MatrixBase<DerivedEB1> &eb1, Eigen::PlainObjectBase<DerivedGrad> &grad)¶

Compute the gradient of the distance between a two lines segments.

Note

The distance is actually squared distance.

Parameters

  • ea0 – [in] The first vertex of the first edge.

  • ea1 – [in] The second vertex of the first edge.

  • eb0 – [in] The first vertex of the second edge.

  • eb1 – [in] The second vertex of the second edge.

  • grad – [out] The gradient of the distance wrt ea0, ea1, eb0, and eb1.
