template<typename DerivedP, typename DerivedT0, typename DerivedT1, typename DerivedT2, typename DerivedGrad>
void ipc::point_plane_distance_gradient(const Eigen::MatrixBase<DerivedP> &p, const Eigen::MatrixBase<DerivedT0> &t0, const Eigen::MatrixBase<DerivedT1> &t1, const Eigen::MatrixBase<DerivedT2> &t2, Eigen::PlainObjectBase<DerivedGrad> &grad)¶

Compute the gradient of the distance between a point and a plane.

Note

The distance is actually squared distance.

Parameters

  • p – [in] The point.

  • t0 – [in] The first vertex of the triangle.

  • t1 – [in] The second vertex of the triangle.

  • t2 – [in] The third vertex of the triangle.

  • grad – [out] The gradient of the distance wrt p, t0, t1, and t2.
