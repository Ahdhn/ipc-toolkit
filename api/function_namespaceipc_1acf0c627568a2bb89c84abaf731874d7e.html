auto ipc::point_edge_distance(const Eigen::MatrixBase<DerivedP> &p, const Eigen::MatrixBase<DerivedE0> &e0, const Eigen::MatrixBase<DerivedE1> &e1)¶

Compute the distance between a point and edge in 2D or 3D.

Note

The distance is actually squared distance.

Parameters

  • p – [in] The point.

  • e0 – [in] The first vertex of the edge.

  • e1 – [in] The second vertex of the edge.

Returns

The distance between the point and edge.
