template<typename DerivedEA0, typename DerivedEA1, typename DerivedEB0, typename DerivedEB1>
auto ipc::edge_edge_distance(const Eigen::MatrixBase<DerivedEA0> &ea0, const Eigen::MatrixBase<DerivedEA1> &ea1, const Eigen::MatrixBase<DerivedEB0> &eb0, const Eigen::MatrixBase<DerivedEB1> &eb1, const EdgeEdgeDistanceType dtype)¶

Compute the distance between a two lines segments in 3D.

Note

The distance is actually squared distance.

Parameters

  • ea0 – The first vertex of the first edge.

  • ea1 – The second vertex of the first edge.

  • eb0 – The first vertex of the second edge.

  • eb1 – The second vertex of the second edge.

  • dtype – The point edge distance type to compute.

Returns

The distance between the two edges.
