Template Function ipc::line_line_distance_hessian¶

Function Documentation¶

template<typename DerivedEA0, typename DerivedEA1, typename DerivedEB0, typename DerivedEB1, typename DerivedHess>
void ipc::line_line_distance_hessian(const Eigen::MatrixBase<DerivedEA0> &ea0, const Eigen::MatrixBase<DerivedEA1> &ea1, const Eigen::MatrixBase<DerivedEB0> &eb0, const Eigen::MatrixBase<DerivedEB1> &eb1, Eigen::PlainObjectBase<DerivedHess> &hess)¶

Compute the hessian of the distance between a two lines in 3D.

Note

The distance is actually squared distance.

Warning

If the lines are parallel this function returns a distance of zero.

Parameters

  • ea0 – [in] The first vertex of the edge defining the first line.

  • ea1 – [in] The second vertex of the edge defining the first line.

  • ea0 – [in] The first vertex of the edge defining the second line.

  • ea1 – [in] The second vertex of the edge defining the second line.

  • hess – [out] The hessian of the distance wrt ea0, ea1, eb0, and eb1.
