template<typename DerivedP, typename DerivedE0, typename DerivedE1, typename DerivedHess>
void ipc::point_edge_distance_hessian(const Eigen::MatrixBase<DerivedP> &p, const Eigen::MatrixBase<DerivedE0> &e0, const Eigen::MatrixBase<DerivedE1> &e1, const PointEdgeDistanceType dtype, Eigen::PlainObjectBase<DerivedHess> &hess)¶

Compute the hessian of the distance between a point and edge.

Note

The distance is actually squared distance.

Parameters

  • p – [in] The point.

  • e0 – [in] The first vertex of the edge.

  • e1 – [in] The second vertex of the edge.

  • dtype – [in] The point edge distance type to compute.

  • hess – [out] The hessian of the distance wrt p, e0, and e1.

A set of reusable functions to integrate IPC into an existing simulation.
