void ipc::construct_constraint_set(const Eigen::MatrixXd &V_rest, const Eigen::MatrixXd &V, const Eigen::MatrixXi &E, const Eigen::MatrixXi &F, const double dhat, Constraints &constraint_set, const Eigen::MatrixXi &F2E = Eigen::MatrixXi(), const double dmin = 0, const BroadPhaseMethod &method = BroadPhaseMethod::HASH_GRID, const bool ignore_codimensional_vertices = false, const std::function<bool(size_t, size_t)> &can_collide = [](size_t, size_t) { return true;})¶

Construct a set of constraints used to compute the barrier potential.

Note

The given constraint_set will be cleared.

Note

V can either be the surface vertices or the entire mesh vertices. The edges and face should be only for the surface elements.

Parameters

  • V – [in] Vertex positions as rows of a matrix.

  • E – [in] Edges as rows of indicies into V.

  • F – [in] Triangular faces as rows of indicies into V.

  • dhat – [in] The activation distance of the barrier.

  • constraint_set – [out] The constructed set of constraints.

  • F2E – [in] Map from F edges to rows of E.

  • dmin – [in] Minimum distance.

  • method – [in] Broad-phase method to use.

  • ignore_codimensional_vertices – [in] Ignores vertices not connected to edges.

  • can_collide – [in] A function that takes two vertex IDs (row numbers in V) and returns true if the vertices (and faces or edges containing the vertices) can collide. By default all primitives can collide with all other primitives.

A set of reusable functions to integrate IPC into an existing simulation.
